# Senior Mobile Engineer

## Responsibilities
- Led the technical implementation of the VIEW app, engineering high-concurrency data processing pipelines for real-time diabetes health monitoring at scale.
- Resolved critical memory leaks and optimized multithreading performance in the Android domain layer, reducing overall memory footprint by 37%.
- Architected low-latency communication modules using Kotlin and Java, ensuring reliable, stateful data synchronization across thousands of active medical devices.
